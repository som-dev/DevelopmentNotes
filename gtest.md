# googletest

## Filters
```
# Run only certain tests
./test --gtest_filter=TestSuite.TestName
./test --gtest_filter=TestSuite.Variation*
./test --gtest_filter=TestSuite.*

# Filter out certain tests
./test --gtest_filter=-TestSuite.TestName
./test --gtest_filter=-TestSuite.Variation*
./test --gtest_filter=-TestSuite.*
```
